Code 128 generator

Enter an EAN-13 barcode and an optional serial / suffix. The generator will build the payload as:

510 + <EAN-13> + <serial> + <Luhn check digit of the whole number so far>

Example format: 510<ean-13>0000011960

Unlock generator

Enter the unlock key to use this generator.

If you already know the key, type it and press Unlock. (Case-sensitive)
